# YouTube Trending Data Analysis 📺

This project analyzes trending YouTube videos to uncover patterns in views, likes, and audience engagement.

---

## 📌 Objective

The goal of this project is to explore trending YouTube data and understand factors that influence video popularity and engagement.

---

## 🛠 Tools & Technologies Used

* Python
* Pandas
* Matplotlib

---

## 📊 Analysis Performed

* Identified top channels with the most trending videos
* Analyzed relationship between views and likes
* Calculated engagement ratio (likes/views)
* Studied impact of upload timing on trending videos

---

## 🔍 Key Insights

* Videos with higher views generally tend to receive more likes, indicating a strong positive relationship.
* Certain channels appear frequently in the trending list, showing consistent performance.
* Engagement varies across videos, meaning high views do not always guarantee high interaction.
* Some videos achieve high engagement despite lower views, indicating strong audience connection.
* Upload timing may influence a video's chances of trending.

---

## 📁 Dataset

The dataset used in this project can be accessed here:
https://www.kaggle.com/datasets/datasnaek/youtube-new

---

## 🚀 Conclusion

This analysis highlights how different factors such as views, engagement, and upload timing affect a video's performance on YouTube. These insights can be useful for content creators and analysts to better understand audience behavior.
